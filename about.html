the people

Created by Sophie True, Alex Jordan, and Thomas Jamrozik for a 9th grade Health class in which our final project for the trimester was to create a Public Service Announcement, or PSA.

We chose to create a website about this issue because we feel that it doesn't get enough attention. People just don't understand that texting while driving is an issue.

the technology

This website uses several awesome technologies. Thanks to Twitter Bootstrap (@mdo and @fat on Twitter), Font Awesome (@fortaweso_me), GitHub and GitHub:Pages, and HTML5 Boilerplate (@h5bp).

Of course, HTML5 Boilerplate comes with several more technologies that this website uses, so thanks to HTML5Shiv (and jQuery, since all the Bootstrap plugins are powered with it!). The Initializr tool was also involved, so thanks to @initializr/@verekia too.
